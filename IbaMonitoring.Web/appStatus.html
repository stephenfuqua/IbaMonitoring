IbaMonitoring.org Application Status

The IbaMonitoring.org data collection website is currently in an archive mode and not accepting new data. For budget reasons, the database is also offline at this time and thus results are not available. For more information on the data results of this project, please see

Homayoun, Tania Z., and Robert B. Blair. "Value of park reserves to migrating and breeding landbirds in an urban important bird area." Urban Ecosystems 19.4 (2016): 1579-1596.
